Sets de datos

Hola, en esta entrada les voy a contar mi experiencia con datos de fuentes abiertas a todo público, provenientes de la página web Kaggle.com vasados en datos recogidos por la Agencia Aeroespacial de los Estados Unidos (NASA).
El archivo de extension .csv muestra una coleccion de datos referentes a avistamientos de ovnis por su traduccion al ingles UFO. Se pretende con este analisis primeramente obtener una data limpia y libre de errores. Posteriormente se intentara predecir nuevos avistamientos a partir de el analisis de las muestras procesadas.

    Date/Time : Muestra el momento exacto del avistamiento.
    Country: Pais CANADA o USA
    City: Ciudad del avistamiento
    Shape: Forma geometrica
    Summary: Relato
    lat: valores de latitud
    lng: valores de longitud

Se recomienda empezar con una limpieza exhaustiva de datos se puede hacer desde scripts de Python o utilizando tecnología de Excel. Utilizo un archivo llamado rechape.py para generar nuevas tablas en formato .csv para el posterior análisis.
